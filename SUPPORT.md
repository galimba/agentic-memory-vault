# Support

## Getting Help

- **Documentation**: Start with the [README](README.md) and files in [docs/](docs/)
- **Discussions**: Use [GitHub Discussions](../../discussions) for questions, ideas, and general conversation
- **Bug Reports**: Use [GitHub Issues](../../issues) with the Bug Report template
- **Feature Requests**: Use [GitHub Issues](../../issues) with the Feature Request template

## Before Asking for Help

1. Read `CLAUDE.md` — it is the comprehensive specification
2. Run `bash .vault/scripts/vault-tools.sh doctor` to diagnose issues
3. Check existing [Issues](../../issues) and [Discussions](../../discussions)

## Response Times

This is an open-source project maintained by volunteers. We aim to respond to issues within one week, but cannot guarantee timelines. Security issues receive priority response (see [SECURITY.md](SECURITY.md)).
